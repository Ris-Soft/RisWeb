# Vue 3 + Vite + Vuetify 美观模板

本项目是一个基于 Vue 3、Vite 和 Vuetify 的美观模板，包含：
- 现代风格导航栏
- 欢迎卡片
- 响应式布局

## 启动项目

```sh
pnpm install
pnpm run dev
```

访问 http://localhost:5173 预览首页。

## 技术栈
- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Vuetify](https://vuetifyjs.com/zh-Hans/)

## 特色
- 使用 Vuetify 组件实现现代美观的 UI
- 响应式布局，适配多端
- 代码结构清晰，便于扩展
